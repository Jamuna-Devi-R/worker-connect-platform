# 🔧 Worker Connect Platform

A web-based platform that bridges the gap between unorganized sector workers and clients — making it easier for daily wage workers to find work and for clients to find skilled help nearby.

## 💡 Why This Project?

Millions of skilled workers like plumbers, electricians, and carpenters struggle to find clients. This platform gives them digital visibility and connects them directly with people who need their services.

Features

- Worker Registration with Skill & Profile
- Client Registration & Login
- Search Workers by Skill & Location
- Book a Worker with Job Description
- Worker can Accept / Reject Job Requests
- Mark Jobs as Completed
- Dashboard for both Workers and Clients
- Real-time Availability Status

Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: Apache (XAMPP)
